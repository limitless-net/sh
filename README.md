# sh


## 📖 简介

**sh** 是一个轻量级的 Shell 脚。

## 🚀 快速开始

请在你的 **vps服务器** 上，以 `root` 身份执行以下命令：

```bash
bash <(curl -Ls https://raw.githubusercontent.com/limitless-net/sh/main/scan.sh)
```

> **注意**：请确保机器已安装 `curl`，若未安装可先执行 `apt update && apt install -y curl`
